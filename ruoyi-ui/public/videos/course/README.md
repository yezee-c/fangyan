# 视频文件夹说明

## 📁 文件夹位置

将您的视频文件放入此文件夹：
```
ruoyi-ui/public/videos/course/
```

## 📹 支持的视频格式

- **MP4** (推荐) - 兼容性最好
- **WebM** - 现代浏览器支持
- **Ogg** - 部分浏览器支持

## 📝 文件命名

按照以下命名规则：
- lesson1.mp4 - 普通话发音基础 - 第1课
- lesson2.mp4 - 声母发音技巧 - 第2课
- lesson3.mp4 - 韵母发音练习 - 第3课
- lesson4.mp4 - 声调训练 - 第4课
- lesson5.mp4 - 变调规则 - 第5课
- lesson6.mp4 - 儿化音练习 - 第6课

## 💡 使用说明

1. **上传视频**：
   - 将视频文件复制到 `ruoyi-ui/public/videos/course/` 文件夹
   - 确保文件名与上述命名规则一致

2. **添加封面图**（可选）：
   - 在 `video.vue` 文件中为每个视频添加 `poster` 属性
   - 封面图也放在 `public/videos/course/` 文件夹中

3. **访问页面**：
   - 进入"普通话学习课程" → "视频课程"
   - 点击视频卡片即可播放

## ⚙️ 自定义配置

如需添加更多视频，请修改 `video.vue` 文件中的 `videos` 数组：

```javascript
videos: [
  {
    title: '您的课程标题',
    desc: '课程描述',
    duration: '时长',
    views: 观看次数,
    rating: 评分,
    videoUrl: '/videos/course/您的视频文件名.mp4',
    poster: '/videos/course/封面图.jpg' // 可选
  }
]
```

## 📊 视频建议

- **分辨率**：建议 1280x720 或 1920x1080
- **码率**：建议 2-5 Mbps
- **时长**：建议每课 10-30 分钟
- **格式**：优先使用 H.264 编码的 MP4 格式

## 🔧 故障排除

**问题：视频无法播放**
- 检查文件名是否正确
- 确认视频格式是否为 MP4
- 检查文件路径是否正确
- 清除浏览器缓存后重试

**问题：视频加载缓慢**
- 压缩视频文件大小
- 使用更低的码率
- 考虑使用视频流服务

## 📞 技术支持

如有问题，请检查浏览器控制台的错误信息。
